Create - элемент запроса который создает таблицу

Insert -элемент запроса, который вставляет данные в таблицу

Select - элемент запроса который определяет выбранные столбцы их порядок и источник данных.

Update - элемент запроса,  который изменяет имеющиеся данные в таблице

Drop - элемент запроса б который удаляет строки и столбцы
