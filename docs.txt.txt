1)Структура данных- это структуры которые хранят информацию в упорядоченном виде.
2) Структуры данных пользуются для хранения связанных данных
3)Hashmaps- это тип структуры данных, которая хранит данные в виде пары ключ-значение.
4) Linked list - это структура данных , которая состоит из элементов связанных узлами
5)linked list- нужен для  того чтобы использовать оперативную память на все 100% то есть простой лист все сохранял по порядку и /
если не находил свободные блоки он не сохранялся а Linked List сохраняет где угодно и просто показывает где находится следующий индекс
Hash Map-нужен чтобы упростить код и написать все за 1 операцию то есть big o будет 1 и нагрузка в систему будет меньше
